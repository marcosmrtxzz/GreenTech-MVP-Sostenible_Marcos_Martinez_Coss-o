## Que he cambiado

- He eliminado el Bootstrap, font Awesome y google fonts.  
- HTML lo he reducido a lo mas importante que es el titulo, el parrafo, el boton y el footer.  
- CSS nativo que es mucho mas ligero.  
- La imagen la he optimizado como fondo en cover en vez de .webp
- Pagina sin scroll es decir que no se pueda utilizar la ruedita y con el contenido centrado.  
- Sin JavaScript, porque no es necesario, ademas no se utilizarlo

## Por que es mas sostenible

- Tiene menos peso.
- Se calienta menos y mas vida util  
- Menos dependencias y por lo tanto menor huella de carbono.  
- Ecodiseño aplicado por optimizar las fuentes...
